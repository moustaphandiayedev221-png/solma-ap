# Page Politique de confidentialité — SOLMA

Cette page HTML peut être hébergée sur **GitHub Pages** pour fournir un lien à Apple dans App Store Connect.

## Contenu

- **Aucune donnée sensible** : pas de clés API, URLs internes ou secrets
- Design moderne, responsive, compatible mobile
- Contact : `moustaphandiayedev221@gmail.com` (modifiable dans `index.html`)

## Hébergement sur GitHub Pages

### Option A : Même dépôt (recommandé)

La page est déjà disponible dans `docs/privacy-policy.html`.

1. **Activer GitHub Pages** : Réglages du repo → **Pages** → Source : **Deploy from a branch**
2. Choisir la branche **main** et le dossier **/docs**
3. Enregistrer

**URL obtenue** : `https://[VOTRE-USERNAME].github.io/[NOM-REPO]/privacy-policy.html`

### Option B : Dépôt dédié

1. Créer un nouveau repo (ex. `solma-privacy`)
2. Y placer uniquement `index.html` (renommer en `index.html` à la racine)
3. Activer GitHub Pages sur la branche `main`
4. URL : `https://[USERNAME].github.io/solma-privacy/`

## Lien pour App Store Connect

Après déploiement, utilisez par exemple :

```
https://[VOTRE-USERNAME].github.io/[NOM-REPO]/privacy-policy.html
```

ou, avec un dépôt dédié :

```
https://[VOTRE-USERNAME].github.io/solma-privacy/
```

Collez ce lien dans **App Store Connect** → Fiche de l'app → **URL de politique de confidentialité**.
